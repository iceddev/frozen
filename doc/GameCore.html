GameCore

new GameCore()

The GameCore class provides the base to build games on.
Source:
  • GameCore.js, line 1
Example
var myGame = new GameCore({
  canvasId: 'myCanvas',
  update: function(millis){
    // do updating of game state
  },
  draw: function(context){
    // do drawing of the game
  }
});

//start the game
myGame.run();

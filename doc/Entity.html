Class: Entity

Entity

new Entity()

This represents a body and shape in a Box2d world using positions and sizes relative to the Box2d world instance.
Properties:
Name Type Description
id String The id in which to reference this object. Also the userData property for box2d bodies.
x Number The x component of the entity's location
y Number The y component of the entity's location
scale Number The scale in pixels per meter in which to represent this Entity in the box2d world
angle Number The current angle that this entity is rotated at
center Object The x and y locations of what box2d considers the enity's center of mass
restitution Number The percentage of force in which the entity will bounce back from another based on its force pre-collision
density Number The two-dimensional density of the entity. Mass / area.
friction Number The amount of friction on th surface of this entity
linearDamping Number The amount of linear velocity the entity should lose over time
linearVelocity Number The velocity in meters/second given to this entity by box2d calculations
angularVelocity Number The angular velocity in radians/second given to this entity by box2d calculations
angularDamping Number The of amount of angular velocity an entity should lose over time
staticBody Boolean If true, the entity does change its position and angle as the result of box2d calculations
fillStyle String The fillStyle to use for the entity's default renderer
strokeStyle String The strokeStyle to use for the entity's default renderer
hidden Boolean (deprecated) whether to render this object
maskBits Number The 16 bit integer used in determining which other types of entities this body will collide with.
categoryBits Number The 16 bit integer used in describing the type that this enitity is for collisions.
groupIndex Number The 16 bit integer used in overiding maskBits and categoryBits for collision detection.
